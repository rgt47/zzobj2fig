# Cell-Level Formatting Specifications for zzo2f

Spec-object constructors for cell, row, and column formatting in the
Typst backend. Parallel in intent to the `o2f_format` family but
consumed by
[`zzo2f()`](https://rgt47.github.io/zzobj2fig/reference/zzo2f.md) via
the `formats` parameter and applied through
[`tinytable::style_tt()`](https://vincentarelbundock.github.io/tinytable/man/style_tt.html).

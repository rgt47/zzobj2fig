# Margin-figure preamble packages (Typst: none required)

Typst handles margin placement via native `#place()` and `#set page()`
directives; no analog to LaTeX's `sidenotes` / `marginnote` packages.
Returns NULL invisibly and emits a message noting this.

## Usage

``` r
zzo2f_margin_packages(method = NULL)
```

## Arguments

- method:

  Retained for API symmetry.

## Value

NULL (invisible).

# Ensure pdflatex is available

Checks for pdflatex and provides installation instructions if not found.

## Usage

``` r
ensure_pdflatex(verbose = TRUE)
```

## Arguments

- verbose:

  Logical. Print status messages. Default is TRUE.

## Value

Logical indicating if pdflatex is available (invisibly).

## Examples

``` r
if (FALSE) { # \dontrun{
ensure_pdflatex()
} # }
```

# Apply a list of zzo2f_format specs to a tinytable object

Apply a list of zzo2f_format specs to a tinytable object

## Usage

``` r
apply_zzo2f_formats(tbl, formats, data)
```

## Arguments

- tbl:

  A `tinytable` object.

- formats:

  A single `zzo2f_format`, a list of them, or NULL.

- data:

  The data frame backing `tbl`.

## Value

Modified `tinytable` object.

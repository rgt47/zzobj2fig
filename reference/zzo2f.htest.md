# zzo2f method for hypothesis tests

zzo2f method for hypothesis tests

## Usage

``` r
# S3 method for class 'htest'
zzo2f(x, digits = 3, ...)
```

## Arguments

- x:

  An htest object.

- digits:

  Number of decimal places.

- ...:

  Additional arguments passed to zzo2f.default.

## Value

Invisibly returns the path to the output file.

# Language directive for Typst (parallel to `babel()`)

Language directive for Typst (parallel to
[`babel()`](https://rgt47.github.io/zzobj2fig/reference/babel.md))

## Usage

``` r
zzo2f_textlang(language)
```

## Arguments

- language:

  Character. An ISO 639 language code (e.g. `"en"`, `"de"`, `"fr"`).

## Value

Character string `"#set text(lang: \"<code>\")"`.

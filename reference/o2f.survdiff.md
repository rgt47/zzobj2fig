# o2f method for survdiff (survival difference tests)

o2f method for survdiff (survival difference tests)

## Usage

``` r
# S3 method for class 'survdiff'
o2f(x, digits = 3, ...)
```

## Arguments

- x:

  A survdiff object from the survival package.

- digits:

  Number of decimal places.

- ...:

  Additional arguments passed to o2f().

## Value

Invisibly returns the path to the generated PDF.

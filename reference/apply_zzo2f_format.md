# Apply a single zzo2f_format spec to a tinytable object

Apply a single zzo2f_format spec to a tinytable object

## Usage

``` r
apply_zzo2f_format(tbl, spec, data)
```

## Arguments

- tbl:

  A `tinytable` object.

- spec:

  A `zzo2f_format` spec.

- data:

  The data frame backing `tbl`.

## Value

Modified `tinytable` object.

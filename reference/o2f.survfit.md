# o2f method for survfit (survival curves)

o2f method for survfit (survival curves)

## Usage

``` r
# S3 method for class 'survfit'
o2f(x, digits = 3, times = NULL, ...)
```

## Arguments

- x:

  A survfit object from the survival package.

- digits:

  Number of decimal places.

- times:

  Optional numeric vector of times at which to report survival.

- ...:

  Additional arguments passed to o2f().

## Value

Invisibly returns the path to the generated PDF.

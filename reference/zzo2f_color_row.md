# Apply background color to rows (Typst backend)

Apply background color to rows (Typst backend)

## Usage

``` r
zzo2f_color_row(rows, background)
```

## Arguments

- rows:

  Integer vector.

- background:

  Character. LaTeX color spec or hex string.

## Value

A `zzo2f_format` spec.

## Examples

``` r
if (FALSE) { # \dontrun{
zzo2f_color_row(c(1, 3, 5), "blue!10")
} # }
```

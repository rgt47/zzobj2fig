# Check Typst dependencies for zzo2f

Checks that tinytable and the Typst CLI are available, providing
installation guidance if not.

## Usage

``` r
check_typst_deps()
```

## Value

A list with availability status for each component (invisibly).

## Examples

``` r
if (FALSE) { # \dontrun{
check_typst_deps()
} # }
```

# Assert parameter meets type requirements

Assert parameter meets type requirements

## Usage

``` r
assert_type(x, name, type, allow_null = FALSE, single = TRUE)
```

## Arguments

- x:

  Value to check.

- name:

  Parameter name for error message.

- type:

  Expected type ("logical", "character").

- allow_null:

  Logical. Allow NULL values.

- single:

  Logical. Require single value.

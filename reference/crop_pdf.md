# Crop a PDF file

Crop a PDF file

## Usage

``` r
crop_pdf(input_pdf, output_pdf, margin = 10)
```

## Arguments

- input_pdf:

  Path to the input PDF file.

- output_pdf:

  Path to the output cropped PDF file.

- margin:

  Numeric. Margin size in points. Can be a single value (applied to all
  sides) or a vector of 4 values (left, top, right, bottom).

# o2f method for matrix

o2f method for matrix

## Usage

``` r
# S3 method for class 'matrix'
o2f(x, rownames = TRUE, ...)
```

## Arguments

- x:

  A matrix to convert to a LaTeX table.

- rownames:

  Logical. Include row names as first column.

- ...:

  Additional arguments passed to o2f.default.

## Value

Invisibly returns the path to the generated PDF.

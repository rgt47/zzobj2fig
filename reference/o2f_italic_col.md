# Italicize specific columns

Convenience function to italicize entire columns.

## Usage

``` r
o2f_italic_col(cols)
```

## Arguments

- cols:

  Integer vector or character vector of column indices or names.

## Value

A o2f_format object with italic=TRUE for specified columns.

## Examples

``` r
if (FALSE) { # \dontrun{
o2f_italic_col("variable")
} # }
```

# Create a geometry package specification

Create a geometry package specification

## Usage

``` r
geometry(margin = NULL, paper = NULL, landscape = FALSE, ...)
```

## Arguments

- margin:

  Page margin specification (e.g., "5mm")

- paper:

  Paper size specification (e.g., "a4paper")

- landscape:

  Logical. If TRUE, sets landscape orientation.

- ...:

  Additional geometry options

## Value

Character string with LaTeX geometry package specification

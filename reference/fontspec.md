# Create fontspec package specification for custom fonts

Create fontspec package specification for custom fonts

## Usage

``` r
fontspec(main_font = NULL, sans_font = NULL, mono_font = NULL)
```

## Arguments

- main_font:

  Main font family name

- sans_font:

  Sans-serif font family name

- mono_font:

  Monospace font family name

## Value

Character vector with LaTeX fontspec package specifications

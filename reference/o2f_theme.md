# Create a o2f theme

Construct a theme object that controls the appearance of tables
generated by o2f().

## Usage

``` r
o2f_theme(
  name = "custom",
  scolor = "blue!10",
  header_bold = TRUE,
  header_color = NULL,
  font_size = NULL,
  font_family = NULL,
  document_class = "article",
  extra_packages = NULL,
  booktabs = TRUE,
  striped = TRUE
)
```

## Arguments

- name:

  Character string. Display name for the theme.

- scolor:

  LaTeX color specification for alternating row shading (e.g.,
  "blue!10"). Set to "white" or use striped=FALSE to disable.

- header_bold:

  Logical. If TRUE, table headers are rendered in bold.

- header_color:

  LaTeX color for header background. NULL for no background.

- font_size:

  LaTeX font size command (e.g., "footnotesize", "small").

- font_family:

  Optional LaTeX font family name. NULL keeps the document's default
  font. NULL for default size.

- document_class:

  LaTeX document class. Defaults to "article".

- extra_packages:

  List of additional LaTeX package specifications.

- booktabs:

  Logical. If TRUE, uses booktabs package for professional horizontal
  rules.

- striped:

  Logical. If TRUE, applies alternating row colors.

## Value

A o2f_theme object (list with class "o2f_theme").

## Examples

``` r
if (FALSE) { # \dontrun{
my_theme <- o2f_theme(
  name = "custom",
  scolor = "green!8",
  header_bold = TRUE,
  font_size = "small"
)

} # }
```
